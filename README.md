# Options Academy

A clear, structured guide to options trading — built for beginners with zero prior knowledge. Covers stocks, calls, puts, buying and selling strategies, with interactive payoff calculators in INR (Indian market context).

## Tech Stack

- **React 18** + **Vite** — fast builds, minimal config
- **Static site** — zero backend, zero runtime cost
- **Deployed on Vercel** — free tier, instant deploys on push

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your GitHub repo
3. Framework preset: **Vite** (auto-detected)
4. Click Deploy — done

Every `git push` to `main` auto-deploys.
